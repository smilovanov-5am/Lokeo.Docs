= Лицевой счёт
:page-toclevels: 4

== Получить количество автоплатежей по лицевому счёта
* GET mobile/contracts/{id}/autoPaymentCount

=== Параметры запроса
* **id** - id лицевого счёта

=== Ответ
psbCardId нужен для использовании карты в автоплатежах ПСБ

.JSON
[%collapsible]
====
[source,json]
----
{
    "count": 1
}
----
====
